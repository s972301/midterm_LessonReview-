# 題目名稱 LessonReview+

#基於通識課的小組提案，製作了一個課程評價網

#使用/操作方式
1. git clone下載
2. npm install 
3. npm start 

#其他說明
使用與參考之框架/模組/原始碼
在前端的製作上，主要是使用及參考了React-bootstrap包及SB Admin2模版的樣式
資料庫的連結使用了express, mongoose, axios包

#我的貢獻
1. 前後端的連結是參考tutotrial自行製作
2. 資料庫文件格式的設計
3. 前端在表格和繪圖有使用一些現成Component,但大部分還是自行製作的
